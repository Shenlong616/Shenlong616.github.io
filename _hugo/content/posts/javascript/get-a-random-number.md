### Returns a random integer

- 0 to 2

```javascript
console.log(Math.floor(Math.random() * 3)); // 0, 1 or 2
```

- 1 to 3

```javascript
console.log(Math.floor(Math.random() * 3) + 1); // 1, 2 or 3
```
