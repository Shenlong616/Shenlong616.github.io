How to merge two dictionaries in Python

July 5, 2022
Table of Contents
  • Python 2.x
  • Python 3.5+

Python 2.x#

x = {"a": 1, "b": 2}
y = {"b": 3, "c": 4}
z = dict(x, **y)

print(z)  # {'a': 1, 'b': 3, 'c': 4}

Python 3.5+#

x = {"a": 1, "b": 2}
y = {"b": 3, "c": 4}
z = {**x, **y}

print(z)  # {'a': 1, 'b': 3, 'c': 4}
