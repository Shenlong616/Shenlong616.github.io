How to merge two dictionaries in Python

Python 2.x x = {"a": 1, "b": 2} y = {"b": 3, "c": 4} z = dict(x, **y) print(z) # {'a': 1, 'b': 3, 'c': 4} Python 3.5+ x = {"a": 1, "b": 2} y = {"b": 3, "c": 4} z = {**x, **y} print(z) # {'a': 1, 'b': 3, 'c': 4}

Multiple sets of kwargs in Python 3.5+

using the ** syntax def process_data(a, b, c, d): print(a, b, c, d) x = {"a": 1, "b": 2} y = {"c": 3, "d": 4} process_data(**x, **y) # 1 2 3 4 process_data(**x, c=23, d=42) # 1 2 23 42
